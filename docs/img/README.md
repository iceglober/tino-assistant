# console screenshots

Add screenshots here when updating the console UI. Reference them from `docs/console.md` as `![label](img/name.png)`.

Conventions:
- Use `tino.example.com` and `123456789012` for hostnames/account IDs in screenshots — never real customer data.
- PNG, max 1600px wide.
- Filename pattern: `<page>-<state>.png` (e.g. `console-compliance.png`, `setup-step-1.png`).
